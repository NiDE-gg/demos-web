# Open Source Notes (Demo)

This folder (`Demo/`) is intended to be publishable.

## Security

- Do not commit production secrets or private credentials.
- If a runtime config is needed later, commit only `*.example` templates.
- Prefer environment variables for sensitive values.

## Contribution Scope

- Keep changes focused to `Demo/` files.
- Test pages in `Demo/pages` and assets in `Demo/style` before merge.

## Disclosure

If a security issue is found in Demo code, report it privately to the maintainer before public disclosure.
